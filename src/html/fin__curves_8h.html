Convert discount factor into zero rate.

Parameters
dfDiscount factor.
yr_fracYear fraction, which is used to convert discount factor into zero rate.
typeType of conversion.
0 => df = 1 / (1 + zr) ^ yr_frac
1 => df = exp(-zr * yr_frac)
Returns
double Zero rate.
